# Enginuity Backend (Node.js + Express)

## Technologies
- REST API
- **AWS S3, RDS, AND EC2** for file storage
- PostgreSQL
- Built with **Express + Node.js**

---

## Prerequisites
- Node.js (>=18) or Docker
- AWS account with:
  - RDS (PostgreSQL)
  - S3 bucket
- `.env` file configured with secrets (see below)
